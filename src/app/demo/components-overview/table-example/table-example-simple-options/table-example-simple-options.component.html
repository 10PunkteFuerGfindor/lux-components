Ermöglicht es, diese Tabelle mit Multiselection-Funktionalitäten zu versehen. Dieses Flag steuert, ob auch ein Klick auf die Zeile die Checkbox aktiviert oder ausschließlich ein Klick auf die Checkbox selbst.
Diese Property enthält den Text der angezeigt wird wenn keine Datensätze gefunden werden konnten. Mit diesem Flag lässt sich das Filter-Inputfeld aus-/einblenden. Diese Property definiert das Label des Filter-Inputfelds.
Diese Property aktiviert/deaktiviert die Pagination für die Tabelle. Hiermit kann die aktuelle Anzahl an Einträgen für die Pagination eingestellt werden. Diese Property enthält ein Array mit den möglichen Optionen für die Anzahl an Einträgen. Diese Property steuert die Anzahl der Buttons im Paginator.
