Tipp

Formcontrols im Filter anpassen

Die Lux-Form-Controls haben standardmäßig einen Label- und einen Hint/Error-Container. Meistens werden die Fehlermeldungen und Hinweistexte im Filterformular jedoch nicht benötigt.
Für diesen Anwendungsfall kann ab Version 13.0.0 das Attribut luxNoBottomLabels bei den Form-Controls genutzt werden. Dabei wird der Hint/Error-Container vom Form-Control entfernt. Mit der Lux-Layout-Form-Row können dann die Zeilenabstände individuell bestimmt werden.
