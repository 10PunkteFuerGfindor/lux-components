Wichtig: In FontAwesome ab v.5 gibt es 4 Icon-Typen:
'fab' (Brand-Icons, kostenlos), 'fas' (Solid-Icons, kostenlos),
'far' (Regular-Icons, kostenpflichtig), 'fal' (Light-Icons, kostenpflichtig)
Diese Property bestimmt das Icon selbst (z.B. 'fab fa-android' für FontAwesome oder 'android' für Material).
Für Font-Awesome-Icons mit 'fa|fas|fab|far|fal' präfixen, Material-Icons ohne Präfix Über diese Property kann die Größe des Icons festgelegt werden.
Die Größen sind relative 'em'-Werte.
Mithilfe dieser Property wird die Farbe des Icons bestimmt.
Die Farb-Werte stammen aus der luxstyles.scss. Diese Property aktiviert eine abgerundete Darstellung des Icons
Ist nur in Kombination mit einer 'luxColor' sichtbar.
Die Property ermöglicht es, die Margins für das Icon festzulegen
(z.B. '4px', '10px, 5em' oder '1px, 2px, 3px, 4px'). Die Property ermöglicht es, die Paddings für das Icon festzulegen
(z.B. '4px', '10px, 5em' oder '1px, 2px, 3px, 4px').
