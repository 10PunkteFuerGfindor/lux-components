Diese Property definiert das Icon, welches in dem Button (linksbündig, wenn zusätzlich auch noch luxLabel gesetzt ist).
Für Font-Awesome-Icons mit 'fa|fas|fab|far|fal' präfixen, Material-Icons ohne Präfix Bestimmt die Farbe des Buttons.
Die möglichen Werte sind 'primary', 'warn', 'accent', 'default' / undefined. Über den LuxComponentsConfigService kann für alle LuxButtons applikationsweit eine Uppercase-Darstellung forciert werden.
