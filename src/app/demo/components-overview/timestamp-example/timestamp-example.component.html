Hier kann ein Datum für das Beispiel eingetragen werden.
Das Beispiel nimmt dann den Timestamp aus dem Date-Objekt.
Dieser Parameter legt den Präfix für die relativen Zeitangaben fest.
Standardmäßig werden die Präfixe 'in' und 'vor' benutzt.
